**#Endpoint Specification**

---

* JWT auth
* UUID for item id
* logging middleware
* GORM
* ECHO

**#Endpoint Feature**

---

- Get all items data
- Get all items data by ID
- Get all items data by CategoryID
- Get all items data by keyword and name
- Delete items
- Create new item
